Humondial — match card design gallery

Inventory of static HTML explorations for the promo PWA match card (team row, live header, guess strip). Deployed next to the Vite build via public/ — URL path /match-card-design/ after Cloudflare Pages deploy.

How many “permutations”?

These are separate design inventories, not one Cartesian grid. Combining them arbitrarily would multiply into layouts that were never mocked.

  • 10 team + flag row faces (#v01 … #v10)
  • 3 shortlisted face directions (A/B/C)
  • 6 live-match card shells
  • 6 variants for showing the bingo guess during live broadcast

Total distinct mock snapshots in this bundle: 10 + 3 + 6 + 6 = 25, across 4 linked pages.
Production MatchCard in src/screens/HomeScreen.jsx implements one cohesive layout with status-driven branches (open / locked / live / final, prediction, venue broadcast rim, etc.).
