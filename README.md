# html-handoffs

Shared static HTML handoffs. Each publish lives under `pub/<uuid>/index.html`.

**Site:** https://tatarco.github.io/html-handoffs/

**Pages:** Repo → Settings → Pages → Source: **Deploy from a branch** → Branch **main**, root (**/**). Do **not** call `gh api .../pages` POST from automation — that attaches managed Actions deployments.
